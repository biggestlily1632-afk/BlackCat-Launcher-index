隱私權政策

最後更新：2025 年 4 月
🐈‍⬛ 黑貓啟動器尊重您的隱私。本軟體不會將任何個人資料上傳至外部伺服器，所有資訊均儲存於您的本機電腦中。

一、收集的資訊

本軟體在您使用微軟帳號登入時，會透過 Microsoft OAuth 取得以下資訊：

  • Microsoft 帳號授權 Token（用於驗證身分）
  • Minecraft 玩家名稱及 UUID
  • Minecraft 存取 Token（用於啟動遊戲）

上述資訊僅儲存於您的本機電腦中（game-data/auth.json），不會上傳至任何伺服器。

二、資訊的使用

所取得的資訊僅用於以下目的：

  • 啟動 Minecraft 遊戲
  • 驗證您的正版資格
  • 自動重新整理登入狀態（refresh token）

三、資訊的儲存與刪除

  • 所有資料均儲存於本機，不經過任何中間伺服器
  • 登出後，相關 Token 將從本機檔案中清除
  • 您可隨時刪除 game-data 資料夾以清除所有資料

四、第三方服務

本軟體在運作過程中會連線以下第三方服務：

  • Microsoft Identity Platform — 帳號登入驗證
  • Mojang / Minecraft API — 正版驗證及玩家資料
  • Modrinth API — 模組、光影、材質包搜尋
  • Adoptium — Java 執行環境下載
  • GitHub API — 版本更新檢查

上述服務均有各自的隱私權政策，請參閱其官方網站。

五、政策變更

我們保留隨時修改本政策的權利。修改後的政策將於本頁面更新，並在 GitHub 發布公告。繼續使用本軟體即表示您同意修改後的政策。

六、聯絡我們

如有任何隱私權相關問題，請透過 GitHub Issues 聯絡我們。
