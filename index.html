黑貓啟動器

簡潔、快速、支援正版驗證的 Minecraft 啟動器
支援 Vanilla、Fabric、Forge、NeoForge 及 Modrinth 模組庫

⬇️ 立即下載 📂 GitHub
✨ 功能特色
一切你需要的，都在這裡
🔐
正版帳號驗證
使用 Microsoft OAuth 2.0 正版驗證，安全登入你的 Minecraft 帳號，無需輸入密碼。
🧩
多載入器支援
支援 Vanilla、Fabric、Forge、NeoForge，一鍵安裝，輕鬆管理不同版本的模組環境。
🔍
Modrinth 模組庫
內建 Modrinth 搜尋，直接搜尋並安裝模組、光影、材質包，依載入器自動篩選。
🧠
自由調整記憶體
每個安裝檔可獨立設定最低與最高記憶體，讓遊戲效能最佳化。
🔄
自動更新
啟動時自動檢查 GitHub Releases，有新版本時跳出提示視窗，一鍵完成更新。
💥
崩潰報告
遊戲崩潰時自動讀取報告，跳出視窗顯示詳細內容，支援一鍵複製與開啟資料夾。
☕
自動下載 Java
根據 Minecraft 版本自動選擇並下載對應的 Java（8 / 17 / 21），無需手動安裝。
🔗
桌面捷徑
一鍵為任意安裝檔建立桌面捷徑，快速啟動你最常玩的 Minecraft 版本。
🐈‍⬛
簡潔介面
暗色系設計，清晰的側邊欄與直覺操作，讓管理多個安裝檔變得輕鬆簡單。
